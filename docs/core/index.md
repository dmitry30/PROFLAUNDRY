# Ядро (Core)

> ⚠️ Этот раздел в проработке. Полный состав ядра ещё не определён.
> Зафиксированы принципы и ключевые связи — конкретный список объектов уточняется.

Ядро — обязательный монолитный блок, который присутствует в любой организации на платформе. Реализован как Django-приложение `platform_core`.

## Что точно входит в ядро

### Инфраструктура платформы
- Организации (Tenants)
- Пользователи и роли (три уровня: платформа / организация / клиент)
- Модули организации (включение/отключение опциональных модулей)

### Базовые справочники
Примерный состав — по аналогии с 1С, уточняется:

| Справочник | Примечание |
|-----------|-----------|
| Контрагенты | Юридические и физические лица, с которыми ведётся деятельность |
| Физические лица | Персональные данные людей (сотрудники, контактные лица, клиенты-физлица) |
| Сотрудники | Привязаны к физическим лицам; имеют роли и права в системе |
| Клиенты | Смысловая единица «кому оказываем услугу/продаём»; у одного клиента может быть несколько контрагентов |
| Валюты | |
| Единицы измерения | |
| Страны | |
| Банки | |
| Банковские счета | Счета контрагентов и организации |

### Базовые документы
Примерный состав — уточняется:

| Документ | Примечание |
|---------|-----------|
| Заказ | Центральный документ — запрос на выполнение работы/поставку |
| Счёт на оплату | Выставляется клиенту |
| Оплата | Регистрация поступившего платежа |

### Константы организации
Настройки уровня организации: основная валюта, реквизиты, параметры учёта.

## Ключевая связь: Клиент и Контрагент

В системе разделены два понятия:

```
Клиент
├── Контрагент А (ООО «Ромашка»)
├── Контрагент Б (ИП Иванов И.И.)
└── Контрагент В (ООО «Лютик»)
```

- **Клиент** — смысловая сущность «кому мы оказываем услуги». Может быть физлицом или представлять группу компаний.
- **Контрагент** — юридическое или физическое лицо для целей документооборота и расчётов.
- Один клиент может иметь несколько контрагентов (несколько юрлиц одного бизнеса).
- Документы (счета, оплаты) оформляются на конкретного контрагента, но принадлежат клиенту.
