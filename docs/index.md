# PROFLAUNDRY Platform

Веб-платформа для управления прачечным бизнесом.

## Концепция

Система позволяет организациям вести полноценный учёт: клиенты и контрагенты, номенклатура услуг, заказы, выставление счетов, расчёты — всё в одном месте, через браузер.

Система многоарендная (multi-tenant): несколько независимых организаций работают в одной системе, каждая в полностью изолированном пространстве данных.

## Три уровня доступа

| Уровень | Кто | Что видит |
|---------|-----|-----------|
| **Платформа** `/platform/` | Администраторы платформы | Все организации, все данные |
| **Организация** `/app/` | Сотрудники организации | Только данные своей организации |
| **Клиент** `/portal/` | Клиенты организации | Только свои данные (заказы, счета, история) |

Каждый уровень имеет отдельный интерфейс входа и изолированный набор экранов.
