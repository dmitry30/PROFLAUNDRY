# PROFLAUNDRY Platform

Веб-платформа для управления бизнесом — аналог 1С:Предприятие в браузере.

## Концепция

Платформа предоставляет организациям инфраструктуру для ведения бизнеса: справочники, документы, регистры — всё как в 1С, но через веб.

Система многоарендная (multi-tenant): несколько независимых организаций работают в одной системе, каждая в полностью изолированном пространстве данных.

## Состав системы

| Часть | Описание |
|-------|----------|
| **Ядро (Core)** | Обязательный монолитный блок. Всё что нужно любой организации: базовые справочники, документы, права доступа. |
| **Модули** | Опциональные блоки функциональности. Каждый модуль — отдельное Django-приложение, реализующее конкретную предметную область. |

## Три уровня доступа

| Уровень | Кто | Что видит |
|---------|-----|-----------|
| **Платформа** | Администраторы платформы | Все организации, все данные |
| **Организация** | Сотрудники организации | Только данные своей организации |
| **Клиент** | Клиенты организации | Только свои данные (заказы, счета, история) |

Каждый уровень имеет отдельный интерфейс входа и изолированный набор экранов.
