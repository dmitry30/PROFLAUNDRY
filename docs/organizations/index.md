# Организации

Этот раздел описывает конкретные организации, подключённые к системе PROFLAUNDRY.

Каждая организация — независимый тенант со своими клиентами, прачечными, сотрудниками и настройками. Документация по организации содержит:
- специфику бизнес-процесса именно этой организации
- конфигурацию (модули, прайсы, роли)
- замечания по миграции или интеграции

## Организации

- [Прачечная CCL](ref:organizations.ccl) — первый подключённый тенант, используется как эталон требований
