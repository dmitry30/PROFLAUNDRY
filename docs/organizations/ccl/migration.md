# Требования к миграции CCL → PROFLAUNDRY

Список требований, которые должны быть реализованы в PROFLAUNDRY для того, чтобы CCL смог полностью перейти со старой системы.

Старая система CCL задокументирована отдельно (проект `ccl` в docs-портале).

*Раздел наполняется по мере анализа расхождений между старой системой и PROFLAUNDRY.*
