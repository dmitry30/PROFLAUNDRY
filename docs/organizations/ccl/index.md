# Прачечная CCL

CCL (Clean Crystal Laundry) — первая организация, подключаемая к PROFLAUNDRY. Служит одновременно как:
- **тестовый тенант** — реальные данные и реальный процесс для проверки системы
- **эталон требований** — функционал PROFLAUNDRY должен полностью покрывать потребности CCL для успешной миграции со старой системы
