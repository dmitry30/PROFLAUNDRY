# Бухгалтерия

Бухгалтерский модуль обеспечивает учёт финансовых операций организации.

## Функции

- Выставление счетов клиентам на основе расчётных листов
- Регистрация входящих оплат
- Учёт движения денежных средств (расходы на закупки, зарплаты и т.д.)
- Аналитика по задолженностям и оборотам

*Детальное описание модуля будет разработано отдельно.*
