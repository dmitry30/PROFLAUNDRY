# Чат

Модуль коммуникации между участниками системы.

## Возможности

- Переписка между сотрудниками организации
- Переписка сотрудников с клиентами
- Контекстные подчаты — переписка в рамках конкретного документа (например, заказа)

## Контексты чата

| Контекст | Описание |
|---------|----------|
| Общий | Между сотрудниками без привязки к документу |
| По заказу | Переписка по конкретному заказу (сотрудники + клиент) |
| По приёмке | Уточнения между прачкой и менеджером |
| Другой | Привязка к любому документу системы |

*Детальное описание системы чата (структура, права, уведомления) будет разработано отдельно.*
