# Номенклатура и прайс-лист

## Номенклатура

Справочник видов изделий, принадлежит организации. Конкретный состав зависит от отрасли — организация заполняет справочник под свою специфику.

Каждая позиция — конкретный вид изделия, который исполнитель считает поштучно при приёмке.

## Группы номенклатуры

Объединение нескольких позиций номенклатуры в одну тарифицируемую единицу.

**Пример:**
```
Группа «Группа А»:
  ├── Позиция А1
  ├── Позиция А2
  └── Позиция А3
```

Группы — часть справочника организации, наравне с номенклатурой. Одна позиция может входить в несколько разных групп в рамках системы, но в прайсе одного клиента — только в одну группу или как отдельная позиция.

## Прайс-лист

Определяет цены для конкретного клиента (или объекта). Строится иерархически:

```
Организация  →  прайс по умолчанию
    ↓ перекрывается
Клиент       →  прайс клиента
    ↓ перекрывается
Объект       →  прайс объекта
```

Действующий прайс определяется по самому конкретному уровню, на котором он задан.

### Строки прайс-листа

Каждая строка — одно из трёх:

| Вид строки | Описание | Пример |
|-----------|----------|--------|
| **Группа** | Набор позиций номенклатуры, тарифицируется по суммарному показателю | Группа А → 120 руб/кг |
| **Позиция** | Отдельная позиция номенклатуры вне группы | Позиция Б → 200 руб/шт |
| **Нестандартная** | Позиция, отсутствующая в номенклатуре, добавляется вручную | Позиция В → 80 руб/м² |

### Единицы тарификации

Единица задаётся для каждой строки прайса отдельно. Может быть любой:

| Единица | Применение |
|---------|-----------|
| за штуку (шт) | Штучные изделия |
| за килограмм (кг) | Изделия, тарифицируемые по весу |
| за кв. метр (м²) | Изделия с площадным измерением |
| другая | По договорённости |

### Ограничение в рамках одного прайса

Позиция номенклатуры в прайсе одного клиента может быть:
- **либо** в одной группе (и тарифицироваться через группу)
- **либо** отдельной строкой

Не одновременно.

## Как прайс применяется в приёмке

При проведении приёмки цены из актуального прайса **фиксируются** в расчётных позициях. Последующее изменение прайса не влияет на уже проведённые приёмки (BP-003).

Исполнитель цены **не видит** — только названия, единицы измерения, количество и вес.
